# Data Science Course at RIT (ISE Department)
Code and IJulia Notebooks for Data Science course

## Dependencies

Run `julia` in your terminal and install the following packages:

```
Pkg.add("IJulia")
Pkg.add("Plots")
Pkg.add("Gadfly")
Pkg.add("DataFrames")
Pkg.add("RDatasets")
```
